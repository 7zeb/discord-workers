discord.workers - An library for discord bots in Workers, requires Wrangler v4

Import https://discord-workers.pages.dev/discord.workers-1.0.0.js, Example to import: import { DiscordBot } from 'https://discord.workers.pages.dev/discord.workers-1.0.0.js'

If you are using slash commands, then just do this: import { DiscordBot, registerSlashCommands } from 'https://discord-workers.pages.dev/discord.workers-1.0.0.js'

If you want to add RPC, please check https://github.com/izhank216/discord-workers/
